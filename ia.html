Using the keyboard

  • Use asterisk ( * ) for multiply ( × )
  • Use slash ( / ) for divide ( ÷ )
  • Use hyphen ( - ) for minus ( − )
  • Use caret ( ^ ) to start exponent ( ↑ )
  • Use backslash ( \\ ) to end exponent ( ↓ )
  • Use percent ( % ) for modulo ( mod )

Grayed-out selections are ignored.
